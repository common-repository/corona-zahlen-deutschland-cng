== Description ==
Corona Zahlen in Deutschland tagesaktuell auf der eigenen Wordpress-Seite.

Features:
* minimierbares Overlay am linken oder rechten Bildschirmrand
* elegantes Widget für Sidebars und Fußleisten
* einbindbares Shortcode-Banner, manueller einsatz im Fließtext
* Auto-Insert funktion für Beiträge und Seiten
	* Der Shortcode wird automatisch an beliebiger Stelle eingefügt
	* getrennte Einstellungen für Beiträge und Seiten 
	* Automatisches Einfügen am Textanfang und/oder Ende.

**This Plugin uses a 3rd Party Service called [Robert Koch-Institut API (v2)](https://api.corona-zahlen.org/docs) by Marlon Lückert to serve the latest Numbers of Infection with Corona in Germany. This service is licensed unter [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/). There is no data actively transfered to the 3rd Party Service except of those, which are technically indispensable. A list of data types beeing transfered can be found [here](https://developer.mozilla.org/en-US/docs/Glossary/Request_header)**

== Installation ==
1. Upload folder `corona-zahlen-deutschland` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. Ready. See more Information in the Settings-Page.
